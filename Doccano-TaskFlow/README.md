# Doccano-TaskFlow
本项目引入了一项高效的定时调度器任务，旨在优化数据管理与协作流程，具体功能涵盖以下三个方面：

1. **自动化数据更新与标注分配**：每日定时执行，将最新的线上业务数据无缝对接至Doccano标注平台，确保数据的时效性。同时，自动分配标注任务给质检人员，实现工作流程的自动化与高效化。
2. **数据回收与对比分析**：同样基于定时机制，自动回收完成质检的数据，并立即启动对比分析程序，生成详尽的数据对比报告。
3. **结果通知与信息共享**：将对比分析得出的关键结果，通过电子邮件形式定时发送给指定人员。
